# 라이브 공고 모니터링 — 2026-06

- 조사일: 2026-06-15
- 범위: 소득무관·B 단독 신청 가능 유형(공공지원/청년안심 민간임대 일반공급, 든든전세) 중심
- ⚠️ 공고 일정·단지는 웹 검색 기반 1차 수집이다. **신청 전 반드시 공식 사이트 원문으로 일정·자격·임대료 확인.**

## 1. 든든전세주택 — LH 6월 수도권 (가장 시의적절, 소득·자산 무심사)

> ⚠️ **정정(2026-06-15):** 최초 작성 시 공고일을 6월로 잘못 기재했다. 실제 출처(MBC 2026-05-18 보도)는 **5월 공고**였고, 수도권 1,253호는 **이미 마감**됐다. 아래로 정정한다.

- 규모: 수도권 1,253호 — **서울 103 · 인천 72 · 경기 1,078** (이미 종료)
- 실제 공고일: 경기 **5/21**, 서울 **5/22**, 인천 **5/28** → **마감 완료**
- 결과: **서울 평균 경쟁률 315:1** (소득·자산 무심사라 수요 집중)
- 현재(6/15) LH 공고중 든든전세는 **전북 군산·경남(지방)뿐**, 수도권 없음 — 사용자 직접 확인
- 다음 기회: LH 예고 **하반기 추가 500호**. 든든전세는 회차로 반복 공급되니 LH청약플러스 알림 설정 권장
- 판정: B 단독이면 소득무관이라 자격은 되나, **서울 315:1 경쟁 + 배점(신생아·다자녀·신혼 우선) 열위** → 단독·무자녀는 당첨 현실성 낮음
- 확인: [LH청약플러스 공고문](https://apply.lh.or.kr/lhapply/apply/wt/wrtanc/selectWrtancList.do?mi=1026) · [마이홈 입주자모집공고](https://www.myhome.go.kr/hws/portal/sch/selectRsdtRcritNtcView.do)

## 2. 청년안심주택 민간임대 — 일반공급 (서울, 소득무관 일반공급)

soco.seoul.go.kr에 6월 추가모집 다수 등록(검색 기준):
- 광흥창역 이랜드PEER신촌 (2026-06-11)
- 길동역 길동생활 A동·B동 (2026-06-04)
- 광운대역 현대프라힐스 (2026-06-04)
- 왕십리역 라봄성동 (2026-06-04)

- 자격: 일반공급(약 80%)은 소득기준 미적용. 민간임대형 임대료 시세 70~85%
- 판정: **B 단독 ✅ (일반공급)** — 단 단지별 자격·임대료·보증금·면적 상이
- 확인: [청년안심주택 모집공고](https://soco.seoul.go.kr/youth/bbs/BMSR00015/list.do?menuNo=400008)
- 액션: 위 단지 중 입지(왕십리·광흥창·광운대) 우선으로 일반공급 잔여·임대료 확인. "추가모집"은 잔여세대라 즉시성 있음.
