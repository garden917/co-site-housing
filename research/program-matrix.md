# 제도 매트릭스 (program-matrix)

- 작성일: 2026-06-15
- 목적: 서울·인천·경기 공공/민간 주거지원 제도의 공식 자격기준을 한곳에 정리하고, 두 사람(A=1주택, B=무주택)에 대한 현재 판정을 표시.
- 판정 코드: ✅ 가능 / 🟡 조건부 / ❌ 불가 / ❓ 추가정보 필요
- 검증수준:
  - **[공고]** 회차 공고문 직접 확인 (SH 2026년 1차 행복주택, 2026-05-28)
  - **[포털]** 공식 포털/공사 안내 웹 확인 (2026-06-15, 마이홈·LH·SH·청년안심주택·뉴홈)
  - **[일반]** 제도 일반기준. 실제 신청 시 해당 회차 공고문으로 재검증 필요
- 핵심 주의: 소득은 **세전 공적자료(건강보험 보수월액 등) 기준**. 두 사람의 보수월액 미확인 상태라 소득 판정은 ❓ 또는 가능성 표기에 그친다. 자산·자동차 상한은 매년/회차 갱신되므로 [일반] 행의 수치는 신청 회차 공고로 확정해야 한다.

## 1. 핵심 판정 요약 (결론 먼저)

2026-06-15 소득 확정 입력(B 1인 월 583만 / 부부 2인 월 1,333만) 반영. 두 사람의 소득이 높아 **소득상한 있는 공공 제도는 매도해도 대부분 ❌**, 실제 경로는 **소득무관 유형**에 집중된다.

- **B가 세대원이거나 1인 세대라면** 월 583만이 모든 1인 청년 소득상한을 초과해 소득제한 청년 공공임대는 ❌. 다만 동생과 2인 세대의 세대주라면 행복주택은 2인 합산 상한 645만을 적용하므로 조건부 가능.
- **부부 합산(2인 1,333만)** 은 가장 관대한 신혼 상한(전세임대Ⅱ 맞벌이 200% ≈ 1,095만)마저 초과 → 미리내집·신혼 전세임대Ⅱ·신혼 특공(공공분양) 등 **매도 후에도 소득으로 ❌**.
- 따라서 실제 경로: ① **소득기준 미적용 일반공급**(공공지원 민간임대/청년안심 민간임대), ② **소득·자산 무관 든든전세**, ③ (매도 후) **일반·민간분양 청약(추첨)** + **(출산 후) 신생아 특례(부부 2억)**.
- **매도의 실익**은 무주택이 필요한 소득무관 상품 + 일반분양 가점 회복 + 신생아 특례 매입에 한정된다. 미리내집 5년 무주택 요건은 소득 차단과 별개로 추가 장벽.

## 3. 제도별 보충 메모 (필요한 조건 변경 + 출처)

### 든든전세주택 — 단기 유력 후보
- 핵심: 소득·자산 심사 없음, 무주택 세대구성원만 요구. 보증금 시세 90% 이하, 최장 8년. 우선순위는 신생아·다자녀 > 신혼·예비신혼 순.
- 두 사람: A 1주택이면 ❌. **B 단독(무주택)** 신청 가능 🟡, 또는 **A 매도 후** 예비신혼/신혼으로 가능. 단 우선순위에서 신생아·다자녀에 밀려 당첨·배정 대기 가능성.
- 필요 변경: (B 단독) 없음 / (부부) A 매도.
- 출처: HUG 든든전세 안내, LH청약플러스 (2026-06-15 확인).

### 공공지원 민간임대 / 청년안심주택 민간임대 — 소득무관 일반공급
- 핵심: 민간임대 **일반공급(약 80%)은 소득기준 미적용**, 무주택 요건도 일반공급은 적용 안 하는 경우가 많음(회차 확인). 특별공급(약 20%)만 청년 120%·신혼 맞벌이 140% 소득기준.
- 두 사람: **B 단독 일반공급 🟡(가장 현실적)**. 임대료가 시세 70~85%로 공공임대보다 높음 → 총주거비 비교 필요. 서울 입지 양호 단지 다수.
- 필요 변경: 없음(일반공급). 단 청약 경쟁·추첨.
- 출처: 서울시 청년안심주택(soco.seoul.go.kr), 국토부 공공지원 민간임대 (2026-06-15 확인).

### 통합공공임대 — 매도 후 중기 후보
- 핵심: 무주택 + 일반공급 중위 150%↓(우선 100%↓), 총자산 3.45억·자동차 3,708만. 최장 30년, 분양전환 없음.
- 두 사람: A 1주택 ❌. **B는 세대구성에 따라 재판정**. 세대원이면 본인 1인 기준으로 초과하고, 동생과 2인 세대의 세대주라면 동생 포함 합산소득으로 판정한다. 매도 후 부부는 합산소득 초과로 ❌.
- 출처: 마이홈포털 통합공공임대 (2026-06-15 확인).

### 장기전세Ⅱ 미리내집 — 매도해도 5년 무주택 요건 주의
- 핵심: 서울 거주 무주택, 혼인 7년 이내 신혼/예비신혼, **부부 모두 공고일 기준 5년간 주택 미소유**. 일반 120%(맞벌이 180%), 우선 100%(맞벌이 150%). 서울 아파트 전세형으로 두 사람 선호와 부합.
- 두 사람: A 1주택 ❌. **매도해도 "최근 5년 내 보유" 이력이 걸리면 즉시 자격 안 됨** → A 주택 취득일·보유기간 확인 필수. 이게 미리내집 가능성의 결정적 변수.
- 필요 변경: A 매도 + 5년 무주택 요건 충족.
- 출처: SH 미리내집 청약자격 안내 (2026-06-15 확인).

### 신혼부부 특별공급(공공분양/뉴홈) — 매도 후 분양 경로
- 핵심: 무주택, 혼인 7년 이내(또는 예비/6세이하 자녀), 우선 소득 100%(맞벌이 120%)↓, 부동산 2.155억·자동차 4,563만, 청약통장 6개월·6회↑.
- 두 사람: 매도 후 무주택 전환 시 🟡. 단 맞벌이 합산소득(연 1.6억대)이 120% 상한을 넘을 가능성 → 일반공급/추첨제·신생아 특공 등 대안 병행 검토.
- 출처: 뉴홈·청약홈·마이홈포털 신혼 특공 (2026-06-15 확인).

### 행복주택 — 세대구성에 따른 조건부
- SH 2026년 1차 공고 직접 확인분. A ❌(1주택), 예비신혼 ❌(1주택·합산소득). B는 세대원이면 1인 상한 4,576,036을 초과해 ❌. 동생과 2인 세대의 세대주라면 2인 상한 6,452,897을 적용하므로 동생 포함 합산소득이 상한 이하일 때 가능.
- 출처: `sources/sh/2026-06-15/` 공고 [공고].

### 저소득 임대(국민·영구·전세임대Ⅰ) — 두 사람 대상 아님
- 도시근로자 70% 이하 등 저소득 기준이라 현재 소득 수준에서 ❌. 회차별 일반공급 여지는 낮음.
