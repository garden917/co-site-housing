# Housing

결혼을 준비하며 서울·인천·경기 지역의 공공주택과 주거지원 제도를 조사하고 의사결정을 기록하는 공간이다.

## 작업 인계

- `CLAUDE_HANDOFF.md`: 현재까지 확인한 조건, 공식 공고 검증 결과, 미확인 사항과 다음 작업 순서

## 현재 조건

- 결혼 계획: 2027년
- 희망 지역: 서울 우선, 인천 및 서울 인접 경기 지역 포함
- 희망 방식: 좋은 입지의 아파트 장기임대, 장기임대 후 매매, 공공분양, 전월세 지원
- 본인: 연봉 약 9,000만 원, 1주택 보유
- B: 연봉 약 7,000만 원, 무주택
- 검토 가능한 선택지: 본인 소유 주택 매도

소득, 자산, 혼인 및 주택 보유 상태는 실제 신청 시점의 공고문 기준으로 다시 확인한다.
